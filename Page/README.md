# Page

Projet de jeu Memory
____________________

Ce programme (nommé Page) contient les fonctionnalités demandées: 
- un compteur de temps restant avec barre de défilement
- une alerte "gagné" ou "perdu"
- un enregistrement des scores en base de données MongoDB (a installer au préalable)
- une implémentation en objet
- une CSS en Less

Lancer le programme sous Visual Studio et il ouvrira Google Chrome (si configuré).
Sinon le compiler par "npm install" et lancer par "node app" puis se rendre
sous Google Chrome à l'adresse localhost:3000. Bon jeu!

Le répertoire source contient également le fichier "Bapteme Morvan Douaud.pdf"
contenant la correction du challenge.
